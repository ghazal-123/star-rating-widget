# star-rating-widget

A simple yet powerful UI component that allows users to provide ratings using a 5-star system. This project highlights the use of CSS pseudo-elements for partial styling (half-stars) and efficient JavaScript event handling.

# ✨Key Features:

- **Half-Star Support:** Advanced CSS implementation using ::after pseudo-elements and overflow: hidden to render half-filled stars.
- **Dynamic Feedback:** Real-time text updates to display the selected rating value.
- **Customizable:** Easily change colors, sizes, or the number of stars via CSS and HTML data attributes.

# 🚀 Technologies Used

**HTML5**: For the structural layout of the calculator.  
**CSS3**: For styling, Flexbox/Grid layouts, and modern UI/UX design.  
**JavaScript**: For the computational logic and DOM manipulation.

# 🛠️ How to Use:

1.Clone the repository: git clone **https://github.com/ghazal-123/star-rating-widget.git**

2.Open the project: Navigate to the project folder and open index.html in your favorite web browser.
